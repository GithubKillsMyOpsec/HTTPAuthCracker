# Getting the code

Firstly get the code:
```
git clone https://github.com/yasserjanah/HTTPAuthCracker.git
```
# Usage
```
cd HTTPAuthCracker/
python3 BasicHTTPAuthCracker.py -t <TARGET> -u <USER_FILE> -p <PASSWORD_FILE>
```
